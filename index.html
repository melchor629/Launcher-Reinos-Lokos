    Launcher Custom de Reinos Lokos.

    Este proyecto intenta crear un Launcher Custom para todo el mundo, que todo el mundo pueda usar. Este launcher incluye un pequeño panel para entrar en las opciones de tu propia cuenta y también una sección con noticias sobre tu server. Este proyecto es muy reciente y todo posible bug seria recomendable postearlos.

    Diseño

    Totalmente diseñado por mi, algunas de las imágenes son de propiedad de Blizard.

    Como ayudar en el proyecto

    Puedes descargar el proyecto con GIT: git clone https://github.com/melchor629/Launcher-Reinos-Lokos.git Para editarlo, necesitas Visual c# 2010. Todos los Pull Requests tienen que tener una descripción de las ediciones.
